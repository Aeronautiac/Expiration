# Expiration Discord Bot

A Discord bot for managing and interacting with the Alumina: Elite community.

## Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/Aeronautiac/Expiration
   ```
2. Navigate to the project directory:
   ```bash
    cd Expiration
    ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Create a `.env` file and fill in your bot token and other required environment variables.

6. Start the bot:
   ```bash
   npm start
   ```
