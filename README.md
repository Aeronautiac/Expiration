# Expiration Discord Bot

A Discord bot for managing and interacting with the Expiration community.

## Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/Aeronautiac/ExpirationJS
   ```
2. Navigate to the project directory:
   ```bash
    cd ExpirationJS
    ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Rename `.env.example` to `.env` and fill in your bot token and other required environment variables.

6. Start the bot:
   ```bash
   npm start
   ```
